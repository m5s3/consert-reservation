## 기본 패키지 구조
```

  도메인/
    controller
      dto/
		  (request, response models)
	  usecase/
		  (Usecase = 각 비즈니스에 맞춰 component 조립)
domain/
	도메인/
		(Models)
		(Components)
		(Repositories)
		infrastructure/
```
